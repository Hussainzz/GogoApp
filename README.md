## Installation
    
`npm install // inside client and server folders`

Next, Populate your env rename `env.example` to `.env`

```
NODE_ENV=
DB_URI = 
JWT_SECRET=
REFERESH_SECRET=
SENDGRID_KEY=
PASSWORD_RESET_EMAIL_TEMPLATE = 
PARTICIPANT_LIMIT=,
TIMEZONE=
API_BASE_URL=
CLIENT_BASE_URL=
REDIS_HOST=
REDIS_PORT=
REDIS_PASSWORD=
```
